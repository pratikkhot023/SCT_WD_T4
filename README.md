# SCT_WD_T4
TASK MANAGER

The Smart To-Do List is a simple and elegant web application for managing daily tasks efficiently. Inspired by the vibrant colors of a peacock, the app features a visually appealing gradient interface with intuitive task management functionalities.
Users can easily add, complete, and delete tasks, with priority-based color coding to help organize work. The app also includes a Dark/Light mode toggle for better accessibility and eye comfort. All tasks are saved in the browserâ€™s local storage, ensuring they persist across sessions without the need for a backend.
This project is ideal for beginners looking to practice HTML, CSS, and JavaScript, as well as anyone who wants a lightweight and aesthetically pleasing to-do list for daily productivity.

Features
Add, delete, and mark tasks as completed
Priority-based color coding: High (Red), Medium (Orange), Low (Green)
Dark/Light mode toggle for user comfort
Tasks persist in browser using local storage
Clean and vibrant Peacock-inspired UI

Tech Stack
HTML5
CSS3 (with gradient and responsive design)
JavaScript (DOM manipulation & local storage)
